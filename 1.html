ПОДРОБНЕЕ Сноски Тестирование выполнено 2 сентября 2020 г. в тестовой лаборатории AMD с целью вычислить среднее значение частоты кадров в 40 играх при разрешении 1920x1080 и высоких настройках графики. R5K-002 Расчеты проводились 18 сентября 2020 г. в тестовых лабораториях AMD для ускорителя AMD Instinct™ MI100 (карта HBM2 PCIe®  32 ГБ)  при пиковой тактовой частоте процессора 1502 МГц. Пиковая производительность вычислений с плавающей точкой составила: 11,54 терафлопс для чисел двойной точности (FP64),  46,1  терафлопс для матрицы одинарной точности (FP32), 23,1 терафлопс  для чисел одинарной точности (FP32) и 184,6  терафлопс для чисел половинной точности (FP16). Согласно опубликованным данным теоретическая пиковая производительность вычислений с плавающей точкой для графического ускорителя  NVidia Ampere A100 (40 ГБ) составила: 9,7 терафлопс для чисел двойной точности (FP64), 19,5 терафлопс для чисел одинарной точности (FP32) и 78 терафлопс для чисел половинной точности (FP16)​.​​ Производители серверов могут вносить в конфигурацию изменения, вследствие чего результаты могут отличаться.  MI100-03​​
